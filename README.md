## Getting Started

To start this project you need to follow this steps:

### Installation

1. Clone the repo
   ```sh
   git clone URL
   ```
2. Install the dependencies
   ```sh
   npm install
   ```
3. Start the project and enjoy!

   ```sh
   npm start

   Go to http://localhost:3000 to view the app in the browser.
   ```
